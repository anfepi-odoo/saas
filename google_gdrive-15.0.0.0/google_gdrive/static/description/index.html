Google drive integration

What does this module solve?

Default behavior

By default, odoo, starting from October 3, 2022, disables integration with Google Drive and

now it is impossible to get an access token out of the box. Previously, inside odoo, a static

odoo application (secret and id) was installed and the user only had to click on the button in the

general settings and give consent to the odoo application. Then it was possible to receive an access token

and interact with the google API. not only disk but also others. Now if you update your Instance it won't

work after October 3, 2022.

Behavior after install

You can set up your own application in the google console and get a json file with settings.

Only with your own application (not odoo) and install it in odoo. The module will automatically

create all the credential and will be available as before the access token, with the get_access_token_gdrive

function of the res.config.settings model. And you will be able to interact with Google API

Features

  • Generate credentials
  • Generate access token (part of credentials)
